License Types

Common Area
$4.00/moExt-to-Ext, 911, Metered Calls (US/CA), Voicemail. One phone or analog device. Ideal for classrooms, lobbies, cafeterias.
Pro
$8.00/moExt-to-Ext, 911, Metered Calls (US/CA), Voicemail, Advanced PBX. Up to 3 phones + mobile app + softphone. For locations or users.
Unlimited
$16.00/moExt-to-Ext, 911, Unlimited Calls (US/CA), Voicemail, SMS/MMS (DID), Advanced PBX. Up to 3 phones + mobile + softphone. For faculty, admins, nurses.
DID
$5.00/moDirect Inward Dial number (phone number). Required for any device or user that needs an external-facing phone number for inbound/outbound PSTN calls.
Basic
FreeExt-to-Ext, 911, Voicemail, Basic PBX. No PSTN access. Mobile app + softphone only. For paraprofessionals, bus drivers, custodians.
